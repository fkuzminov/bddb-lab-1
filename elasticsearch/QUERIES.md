### Индекс finance с документами следующей структуры:
- customer_id — идентификатор клиента
- customer_name — имя клиента
- account_id — идентификатор счёта
- account_type — тип счёта (например, savings, checking)
- balance — текущий баланс
- branch — филиал банка
- transaction_id — идентификатор транзакции
- transaction_date — дата транзакции
- transaction_type — тип транзакции (deposit, withdrawal, transfer)
- amount — сумма транзакции
- loan_id — идентификатор кредита
- loan_amount — сумма кредита
- loan_status — статус кредита (active, closed, overdue)


### Запросы:
1. Получить все документы клиентов из определенного города.
2. Найти все транзакции по конкретному счету.
3. Определить клиентов, у которых баланс превышает заданное значение.
4. Получить список транзакций за определенный период времени.
5. Найти все кредиты со статусом «active».
6. Определить суммарный баланс по каждому филиалу.
7. Получить количество транзакций каждого типа.
8. Найти клиентов, оформивших более одного кредита.
9. Определить максимальную сумму транзакции по каждому типу.
10. Получить средний баланс по типам счетов.
11. Найти клиентов, у которых общая сумма транзакций превышает определенное значение.
12. Определить распределение кредитов по статусам.
13. Получить список клиентов и их место в рейтинге по сумме транзакций.
14. Найти транзакции, сумма которых выше среднего значения по всем транзакциям.
15. Определить последовательность транзакций клиента с указанием текущей и следующей суммы.
