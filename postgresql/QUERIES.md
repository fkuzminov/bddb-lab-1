### Таблицы:
- students (id, full_name, group_id, enrollment_year)
- groups (id, group_name, faculty_id)
- faculties (id, faculty_name)
- courses (id, course_name, credits)
- enrollments (id, student_id, course_id, grade)
- lecturers (id, full_name, department)
- teaching (id, lecturer_id, course_id, semester)


### Запросы:
1. Составить запрос для получения списка студентов с указанием их учебных групп и факультетов.
2. Составить запрос для отображения всех курсов и количества студентов, записанных на каждый из них.
3. Составить запрос для получения списка студентов, у которых средний балл выше определённого значения.
4. Составить запрос для отображения курсов, которые читаются более чем одним преподавателем.
5. Составить запрос для получения списка факультетов и количества студентов в каждом из них.
6. Составить запрос для отображения студентов, не записанных ни на один курс.
7. Составить запрос для получения списка преподавателей и количества различных курсов, которые они ведут.
8. Составить запрос для отображения студентов, имеющих одинаковые оценки по одному и тому же курсу.
9. Составить запрос для получения списка курсов, которые не имеют записанных студентов.
10. Составить запрос для отображения студентов, зачисленных в один и тот же год, с указанием их групп.
11. Составить запрос для получения списка курсов с указанием суммарного количества кредитов по факультетам.
12. Составить запрос для отображения студентов, имеющих наивысшую оценку по каждому курсу.
13. Составить запрос для получения списка преподавателей и студентов, связанных через курсы, которые они ведут и посещают.
14. Составьте запрос, который для каждого студента отобразит его оценки по курсам в хронологическом порядке (по возрастанию идентификатора записи об успеваемости), а также покажет следующую оценку этого же студента в отдельном столбце.
15. Составить запрос для получения списка студентов с указанием их места в рейтинге по среднему баллу внутри своей группы.
