### Коллекции:
- products: { _id, name, category_id, price, stock }
- categories: { _id, category_name, parent_id }
- customers: { _id, full_name, email, registration_date, country } (country необходимо для запроса 15)
- orders: { _id, customer_id, order_date, total_amount }
- order_items: { _id, order_id, product_id, quantity, unit_price }
- suppliers: { _id, name, country }
- shipments: { _id, order_id, shipment_date, status }
- product_suppliers: { _id, product_id, supplier_id } (нет в оригинальном задании, но необходима для запросов 4, 7, 13)


### Запросы:
1. Получить список товаров с указанием их категорий.
2. Определить количество товаров в каждой категории.
3. Найти клиентов, совершивших заказы на сумму выше определённого значения.
4. Определить товары, которые поставляются более чем одним поставщиком.
5. Получить список категорий и количество товаров в каждой из них.
6. Найти клиентов, которые не сделали ни одного заказа.
7. Определить поставщиков и количество различных товаров, которые они поставляют.
8. Найти заказы, содержащие одинаковые товары в одинаковом количестве.
9. Определить товары, которые ни разу не были заказаны.
10. Получить клиентов, зарегистрированных в один и тот же месяц, с указанием их заказов.
11. Определить суммарный объем продаж по категориям.
12. Найти клиентов, сделавших самый дорогой заказ в каждой категории товаров.
13. Получить поставщиков и клиентов, связанных через товары, которые они
поставляют и покупают.
14. Определить товары, по которым средний объем продаж ниже определённого значения. 
15. Получить клиентов с указанием их места в рейтинге по сумме заказов внутри страны.
